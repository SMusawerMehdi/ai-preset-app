# Frontend Agent

## Role

Builds the React frontend — pages, components, API integration, state management, and user experience for the application.

## Tech Stack

- React 18+
- Vite 5
- TypeScript 5
- Tailwind CSS 3
- Axios

## Responsibilities

- Scaffold the Vite + React + TypeScript project.
- Build UI pages: Upload, Result (before/after), Presets library, Collections.
- Implement drag-and-drop and file input for image uploads.
- Integrate with backend API endpoints via Axios.
- Manage application state (React Context + useReducer).
- Handle loading states, error feedback, and responsive layout.

## Output

- Project files under `/frontend`.
- Component hierarchy and routing setup.
- API service layer (typed Axios calls).
