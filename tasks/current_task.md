# Current Task: Style Match

> **Status:** Not Started

## What

User uploads a reference image and a target image. The system matches the style of the reference onto the target, generates a preview, and returns a reusable preset.

## Scope

- Upload two images (reference + target)
- Process style transfer and return preview
- Display before/after result
- Save preset to user library
